All images acquired for this website were gotten off of Google Images, and were "Labeled for non-commercial reuse with modification".

All writing, coding, and design was done by William Andersen and Emmett McDow.

Tools Used

  • Atom was the primary text editor used in the front-end coding of the webpage.

  • Node Package Manager was used to acquire and apply various Javascript libraries and utilities.

  • This entire website was coded using HTML5 and CSS3.

  • Any images on the site were either edited or made using Adobe Photoshop and Illustrator.

  • Owl was used to make the carousel on the homepage.

  • The website W3 Schools was used as a reference and as a source for various portions of the site, such as the navigation bar, parallax scrolling, and modal pop-ups.
